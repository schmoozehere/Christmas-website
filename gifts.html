DIY Gift Ideas

Custom Photo Album

Materials needed: Photo album, photos, decorative paper, stickers

  1. Select your favorite photos
  2. Arrange them chronologically
  3. Add decorative elements
  4. Write personal captions

Handmade Candles

Materials: Wax, wicks, containers, essential oils

  1. Melt the wax
  2. Add fragrance
  3. Pour into containers
  4. Let cool and set

Knitted Scarf

Materials: Yarn, knitting needles, scissors

  1. Choose soft yarn
  2. Cast on stitches
  3. Knit in pattern
  4. Add fringe if desired
